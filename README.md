# lift-simulation
Simulasi sistem kerja lift gedung GKU 1 ITB menggunakan Python dengan logika pergerakan otomatis, manajemen penumpang, dan pembatasan kapasitas.

## Cara Menjalankan
1. Jalankan file `lift_simulation.py`
2. Masukkan panggilan lift sesuai instruksi di terminal
3. Ikuti simulasi hingga selesai
